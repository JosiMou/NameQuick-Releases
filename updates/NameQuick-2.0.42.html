NameQuick 2.0.42 — Reliable renames on macOS 26.1

Release date: 2025-11-10

What’s fixed - Fixed a rare issue where some files couldn’t be opened or renamed after using NameQuick on macOS 26.1. - Files renamed with NameQuick now behave normally in Finder and other apps.
