Release date: 2026-05-14

New

Improvements

  • Improved license recovery after reinstalling NameQuick. Self-Managed licenses can now reconnect to the same Mac more reliably instead of getting stuck on device activation.
  • Made trial activation more reliable. NameQuick now checks existing account access more consistently before asking you to start checkout again.

Bug Fixes

  • Fixed Self-Managed trial activation getting stuck. Existing Self-Managed trial users should now be able to activate with their email instead of being blocked at the license screen.
  • Fixed subscription trials looping back to checkout. Active trial accounts should now unlock from their existing account access even when checkout confirmation is delayed.
  • Fixed device activation recovery for Self-Managed licenses. Reinstalling NameQuick on the same Mac should no longer try to use an extra device slot when the original activation already belongs to that Mac.
  • Fixed the customer portal link in activation recovery. The recovery action now opens the signed-in customer portal instead of sending you to a generic page.
  • Improved the message shown when an email is not found. NameQuick now gives clearer next steps for trying another email, starting a trial, or contacting support.
