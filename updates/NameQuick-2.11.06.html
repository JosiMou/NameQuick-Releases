Release date: 2026-01-22

Improvements

  • **Local text-only models now work with text PDFs.** Ollama and LM Studio text-only models can process PDFs that contain extractable text. Clear error messages appear when a vision model is needed for images or scanned PDFs.
  • **Quick access to support email.** Help menu and Settings now include direct links to reach support.

Bug Fixes

  • **SVG files process correctly for subscription users.** Fixed an issue where SVG, AI, and EPS files failed when processed through the subscription path.
  • **Rules action editors stay visible.** Tag and comment action editors no longer disappear unexpectedly. Archive folder picker now correctly shows the base folder with relative path.
