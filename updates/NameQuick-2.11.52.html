Release date: 2026-06-27

This update improves upgrade flows and recovery when automatic folder processing is interrupted.

New

Improvements

Bug Fixes

  • Fixed paid upgrades from trials getting stuck. Upgrading from a trial to a paid Managed plan now completes more reliably and refreshes credits sooner.
  • Fixed Self-Managed upgrade sheets closing unexpectedly. Upgrading from a Self-Managed trial now keeps the upgrade sheet open when your trial state refreshes.
  • Fixed onboarding license-step loops. The onboarding flow no longer keeps reopening the same license step while license status updates are settling.
  • Fixed interrupted Watch Folder scans. NameQuick now recovers abandoned folder catalog scans after restart and clears stalled processing state more safely, so automatic folder work and content search can recover instead of staying stuck.
  • Fixed repeat retries for files that need review. Unsupported, unreadable, or too-large files are now treated as file issues to review instead of being retried as temporary processing failures.
