Release date: 2026-06-22

This update refreshes Managed plan details, makes queued processing more reliable around setup and access checks, and improves account-access guidance when a license is missing the expected entitlement.

Improvements

  • Updated Managed credit tiers. In-app Managed plan details now align with the current 500, 2,000, 5,000, and 10,000 credit options, so plan and trial screens match the current subscription ladder.
  • Clearer self-managed wording. Trial, upgrade, and account surfaces now use simpler wording around self-managed access and Managed subscriptions.
  • Better queued-processing recovery. Temporary broker or provider rejections now stay retryable instead of looking like final processing failures.
  • Clearer setup and access stops. Jobs that need setup, license access, or policy attention are separated from ordinary retryable work so the queue reports the right next step.

Bug Fixes

  • Fixed false completions in queued processing. NameQuick no longer treats blocked queued work as finished when provider access or retry state still needs attention.
  • Fixed terminal policy skips. Policy-skipped jobs now stop cleanly instead of being retried as if the issue were temporary.
  • Fixed missing-entitlement guidance. Account checks now show the right next step when a customer record does not include the expected self-managed entitlement.
