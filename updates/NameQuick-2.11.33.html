Release date: 2026-05-16

New

  • A clearer AI provider settings experience. The AI settings page now highlights the active provider, available models, setup actions, and local processing choices in one place.
  • Better recovery help for local AI providers. When LM Studio, Ollama, or MLX needs attention, NameQuick now shows focused next steps and support options instead of leaving the problem hidden in the status bar.
  • Diagnostics export for support. You can now create a support-ready diagnostics report when a processing issue needs a closer look.

Improvements

  • Made MLX model loading easier to understand. Downloaded MLX models now ask you to load the model, while runtime installation remains a separate setup step.
  • Improved local model download cards. Model download and install states are easier to scan, with quieter technical details and clearer actions.
  • Improved custom prompts with local AI providers. LM Studio and Ollama now follow a more reliable response format, so custom instructions are less likely to appear in final filenames.
  • Made local provider limits clearer. NameQuick now explains when a local model cannot handle a file instead of showing unrelated credit or usage messaging.
  • Reduced visual noise in the AI provider list. Status labels and action buttons now focus on what needs your attention.

Bug Fixes

  • Fixed MLX showing the wrong action for downloaded models. Downloaded MLX models now show Load model instead of Start server.
  • Fixed confusing MLX setup states. NameQuick now keeps MLX download, runtime install, loading, and ready states separate, so the settings screen no longer suggests reinstalling a runtime when a model only needs to load.
  • Fixed local AI issues appearing as credit problems. MLX, LM Studio, and Ollama failures now stay tied to the selected local provider.
  • Fixed settings window flashes on first open. Settings should open more smoothly without a brief misplaced background or scroll area.
  • Fixed filename recovery for bad local AI responses. NameQuick now catches more unusable filename suggestions and guides you to rerun with clearer instructions.
