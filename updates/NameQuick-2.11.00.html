Release date: 2025-12-18

What's New

  • **Try NameQuick Free with Your Own API Key** – New users can now try NameQuick free for 14 days using their own OpenAI, Anthropic, Gemini, or Ollama API key. The trial includes a set number of AI-powered renames to help you evaluate the app before committing.
  • **Subscription Trials for New Customers** – If you prefer not to use your own API key, you can now start a free trial of our managed AI service. Existing customers returning after a break will see a welcome-back flow that recognizes their account.

Improvements

  • **Faster, More Memory-Efficient Image Processing** – Renamed files containing images now process more efficiently with lower memory usage. Large batches of screenshots and photos should no longer cause slowdowns or high memory consumption.
  • **More Reliable Gemini Integration** – Files that Gemini cannot process (due to content restrictions) now display a clear error message instead of retrying silently. Credits are only charged when a rename actually succeeds.

Fixes

  • Fixed an issue where upgrading from a trial to a subscription didn't immediately reflect in the app, showing stale "trial" indicators until restart.
  • Fixed an issue where purchased BYOK (Bring Your Own Key) licenses were incorrectly shown trial-related UI elements.
  • Trial periods now use server-verified time to ensure accurate expiration regardless of system clock settings.
  • Fixed unnecessary complexity in AI responses by only including the internal rule log field when templates actually use transformation rules.
  • Fixed a template bug where previously working templates broke. These should be stable now.
