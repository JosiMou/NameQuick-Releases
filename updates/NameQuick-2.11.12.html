Release date: 2026-02-05

New

  • **OpenRouter is now available as a BYOK provider.** Bring your own OpenRouter API key to access hundreds of models from a single provider.
  • **Presets sort alphabetically.** The Presets list (formerly "Custom Prompts") now sorts A–Z by default for easier browsing.

Improvements

  • **Japanese and CJK filenames no longer get mangled.** Smart rename no longer inserts separators between every character in CJK output.
  • **Expired BYOK trials stay usable.** An expired trial no longer hard-blocks the app — you can still browse files and explore before upgrading.
  • **Faster, more reliable startup.** The processing queue now waits for licensing to settle before draining, eliminating false "not licensed" errors on launch.
  • **Template test-and-apply is more reliable.** Applying a naming template from the preview now respects rename filters and handles security scopes correctly.

Bug Fixes

  • **Keychain lock no longer stalls processing.** If your Mac's Keychain is locked, NameQuick now shows a clear message and defers work instead of silently failing.
  • **Legacy Keychain entries cleaned up on key removal.** Deleting an API key now removes older duplicate entries that could cause ghost authentications.
  • **Drag-and-drop recognizes existing files again.** Dropping a file that's already in the library no longer creates a duplicate.
  • **Ollama base URL saved correctly.** Custom Ollama endpoints are no longer silently reset.
  • **HUD notifications no longer loop.** Fixed a re-entrancy issue that could cause the notification banner to flash repeatedly.
  • **License status in About window is accurate.** The About panel now reads from the canonical license source instead of a stale cache.
  • **Checkout identity preserved across sessions.** Your customer identity no longer resets when switching between subscription and BYOK flows.
  • **Trial status pill loads on resume.** Reopening the app no longer shows a stale credit count.
