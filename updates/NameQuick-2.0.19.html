v2.0.19 — Create Your Own Prompts

Your naming rules, automated.

You can now write custom prompts for how you want files renamed. Pick one from the toolbar, and NameQuick will use it for every file until you change it. They also work with watch folders for automatic renaming.

Creating a prompt Settings → AI → Manage Custom Prompts. Write your instructions like “date first, then project name” and save. It’ll show up in your toolbar dropdown.

Using prompts The toolbar dropdown now has three sections: Smart Rename, Templates, and Custom Prompts. Pick a custom prompt and the Play button uses it. Switch back to Smart Rename anytime.

Auto-rename with watch folders Click on the Can u → Prompt → pick your custom prompt. New files dropped there automatically rename using that prompt. The folder chip shows which prompt is active.

How it works: - Play button: Uses the prompt shown in the toolbar - Watch folders: Use their own assigned prompts (independent of toolbar) - Templates: Override prompts when selected

Simple. Your file naming system, running on autopilot.

Fixes - Large media drag & drop: Provider‑aware limits—Gemini supports up to 500 MB (with a quick confirm at 100–500 MB); OpenAI audio above 25 MB is blocked with a helpful tip. - Templates: Fixed {original} not resolving. File-based placeholders now work correctly in template renames ({original}, {extension}, {parent}, {size}, {dimensions}).

Known bugs - Drag & drop not processing the same file.
