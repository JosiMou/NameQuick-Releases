Release date: 2026-06-26

This update makes Watch Folders more reliable after folder changes and makes MLX and Keychain setup recovery clearer.

New

Improvements

  • Clearer MLX runtime setup. When a selected MLX model is ready but the runtime still needs installation, Settings now shows an inline setup prompt with an Install MLX runtime action.
  • Clearer empty Watch Folder state. The sidebar now says No folders when no watched folders are configured, and file locations are easier for help and accessibility tools to identify.

Bug Fixes

  • Fixed stale Watch Folder files appearing in All Files. Removing, resetting, or repairing Watch Folders now keeps the library, search index, and selection counts aligned so old tracked items do not keep showing up.
  • Fixed Watch Folder views falling back to the wrong scope. Folder-based views now stay limited to the active watched folders instead of accidentally showing the broader library after stale selections.
  • Fixed Keychain retry buttons for locked keychains. Retry now opens the macOS unlock prompt instead of repeating a check that could never unlock the keychain.
