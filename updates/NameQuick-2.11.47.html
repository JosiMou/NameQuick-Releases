Release date: 2026-06-21

This update adds more practical starter templates and rules, improves template onboarding, and makes queued processing clearer when setup is needed before work can continue.

New

  • More practical template presets. New starters for bank statements, invoices, receipts, admin forms, travel bookings, insurance documents, and contracts make it faster to build repeatable naming workflows.
  • More focused rule templates. Rule starters now cover common document routing needs like statements, invoices, receipts, admin documents, insurance files, travel bookings, and contracts.

Improvements

  • Better template onboarding. The template lesson now shows the full preset catalog, so you can start from the workflow that matches your files instead of choosing from a shortened list.
  • Cleaner starter filenames. Built-in template presets now use more focused fields and consistent separators, making generated names easier to scan and sort.
  • Smarter rule-writing prompts. Chat starters for rules now suggest practical post-rename automations, such as moving invoices, receipts, admin documents, and contracts into the right folders.
  • Clearer queued-job status. Background processing now distinguishes setup problems, such as missing API keys or local models, from ordinary failures so you can see what needs attention.

Bug Fixes

  • Fixed queued processing around setup problems. Jobs that need provider setup, license access, or fresh credits now pause more predictably instead of turning into unclear failures.
  • Fixed review handling for queued renames. NameQuick keeps queued rename work better aligned with review and access checks before processing continues.
