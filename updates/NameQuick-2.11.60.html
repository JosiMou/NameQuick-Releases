Release date: 2026-07-20

This update introduces Connected Inboxes, so invoice and document PDFs can move from approved email senders into NameQuick automatically. It also adds a lifetime Stats dashboard and makes the file library easier to scan and filter.

New

  • Turn incoming email attachments into organized files. Connect an inbox, choose which senders to trust, and route their PDFs into a watched folder where NameQuick can rename them automatically. You can review new senders, pause an inbox, change its destination, and see what was filed.
  • See what NameQuick has done for you. The new Stats dashboard shows your lifetime renames, estimated time saved, recent activity, and how your files were renamed.

Improvements

  • Clearer file status at a glance. The file list and grid now distinguish files that are waiting, processing, ready to apply, renamed, or need attention, so proposed changes are easier to tell apart from completed renames.
  • The same status filters in every folder. Use All, Renamed, and Untouched while browsing any watched folder, not only All Files.
  • A simpler start from an empty library. Choose files or add a watched folder directly from the empty state, with less visual clutter between you and the next action.
  • More useful details when something goes wrong. Failed and unreadable files keep their relevant context and open focused details, making the next step easier to understand.

Bug Fixes

  • Fixed checkout opening in the wrong language. Checkout now follows the language selected in NameQuick and falls back to English when that language is unavailable.
  • Fixed rename suggestions looking already applied. Files with a proposed name now show Ready to apply until the rename actually completes.
