NameQuick v2.0.14

This one focuses on smooth, reliable background processing and better document support — plus a new option for BYOK folks: Claude (Anthropic).

Highlights

  • New: You can now rename Word, PowerPoint, and Excel files (DOCX, PPTX, XLSX).
  • BYOK: Claude (Anthropic) joins the lineup with native PDF + image analysis and explicit model selection.

New

  • Word (.docx): Smart, clean names for your documents — fast previews included.
  • PowerPoint (.pptx): Uses slide text to suggest clear context-aware names.
  • Excel (.xlsx, .xlsm): Concise, informative names for your spreadsheets.
  • Quiet background mode: Keep NameQuick tucked away — it keeps watching your folders and renaming files.
  • BYOK: Added Claude (Anthropic) as a provider — native PDF support via Messages API, image analysis, explicit model selection, key validation, and per‑item cost shown in recent history.

Improvements

  • Folder recovery: If your Mac sleeps or a drive disconnects, NameQuick resumes and catches up automatically.
  • Startup experience: Files already in your watch folders appear instantly. Only new or changed files auto‑process (based on your settings).
  • UI: Cleaner, calmer loading indicator in the grid.

Fixes

  • Automatic processing could pause for some folders.
  • Files added later to a watched folder now process as expected.
  • PPTX files were sometimes flagged as unsupported — they now work consistently.

Thanks for using NameQuick and for all the feedback — it helps a ton.
