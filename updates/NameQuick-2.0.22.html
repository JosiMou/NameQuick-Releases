v2.0.22 — Fresh UI for macOS Tahoe

What’s New

  • Redesigned for macOS Tahoe – Beautiful new user interface optimized for Apple’s latest macOS, with modern visual refinements throughout the app.
  • Vector artists rejoice – Drop in AI, SVG, or EPS files and NameQuick will import, preview, and rename them just like your photos.
  • Templates that feel native – The Templates screen now looks and behaves like Finder. Quick actions sit right where you expect them, and the pencil button jumps straight into editing.

Bug Fixes & Improvements

  • Memory leak fixed – Eliminated a critical memory leak that could cause crashes during extended use—the app now runs reliably for hours without issues.
  • Remembers your style – Fixed persistence issues—separators, prompts, and other preferences now properly stick between sessions.
  • Brackets now allowed – Fixed filename sanitization—you can now use bracket characters in generated filenames without them being stripped out.
  • Custom prompts persist – Fixed custom prompt overrides not being saved—your selected prompts are now properly restored when you restart the app.
  • Clearer UI controls – Added visible Done button and Cancel/Save buttons to the custom prompt manager—no more relying on the Escape key to close dialogs.
  • Visible dialog buttons – Fixed invisible button issue—all confirmation dialogs now show clear Cancel and action buttons (like the cache-clear dialog).
  • Instant template edits – Fixed template editing behavior—tap the pencil to open the editor, and rename from the context menu when you really mean it.
  • Automatic upgrades – Fixed watch folder compatibility—existing watch folders pick up vector support automatically without extra setup.
  • Stability improvements – Squashed numerous bugs affecting app stability and performance.
  • UI polish – Fixed various interface glitches and visual inconsistencies throughout the app.

Why You’ll Love It

  • Broader format support means fewer workflow detours.
  • Smarter AI helps you finish renaming faster.
  • A more native interface keeps everything familiar and simple.
  • Built‑in safeguards protect your large creative files without any extra effort.
