NameQuick 2.0.34

Features

  • Added a global Clean filenames toggle under Preferences → General & Permissions → Defaults. When enabled, NameQuick transliterates German umlauts (ä→ae, ö→oe, ü→ue, Ä→Ae, Ö→Oe, Ü→Ue, ß→ss) and keeps only letters, numbers, single spaces, underscores, and dashes across Smart Rename, Templates, and Custom Prompt flows.

Improvements

  • Updated the rename preview and documentation to call out the exact cleanup rules so teams know what stays and what goes before uploading files.

Fixes

  • Replaced the credit usage queue with a GRDB-backed worker that keeps retries off the main thread, prevents runaway JSON re-encoding, and eliminates the high-CPU watchdog terminations some users saw when Polar was slow.

Cleanup

  • Removed the deprecated Smart Rename and Basic settings tabs now that the Craft-style sidebar covers those options.
