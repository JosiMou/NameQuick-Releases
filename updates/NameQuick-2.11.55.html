Release date: 2026-07-01

This update makes first-run renaming easier to try, improves Quick Process and sidebar moves, and fixes several cases where processing could get stuck or retry when it should ask for review.

New

  • Drop-to-rename first run. New installs now put file dropping and rename previews front and center, so you can try NameQuick by dropping files before setting up folders or automations.
  • Undo support for sidebar folder moves. Files moved by dropping them onto sidebar folders now appear in Recent Renames and can be moved back from the same history flow.

Improvements

  • Clearer All Files empty state. The empty library view now shows a clearer drop card, a Choose Files action, and quick paths to add a watch folder or open Presets.
  • Safer Quick Process drops. Quick Process now makes it clearer when a staged batch is being worked on from disk only, and warns before replacing an existing batch with a new drop.
  • Simpler onboarding. New users now move through setup with passive tips instead of a forced activation tour or extra completion toggles.
  • More dependable notifications. System notifications now request permission at the right time, group related updates more consistently, and open the relevant file, model settings, or Review screen when you click an action.
  • Better BYOK setup checks. Saved provider keys are checked during onboarding, so returning Self-Managed users can continue without reopening Settings.

Bug Fixes

  • Fixed some files getting stuck on “Renaming…”. Unreadable, protected, or duplicate files now resolve to Review or retry cleanly instead of leaving a batch in progress.
  • Fixed Self-Managed checkout setup getting stuck after payment. NameQuick now recognizes the current Self-Managed purchase benefits while keeping older licenses supported.
  • Fixed spreadsheet custom prompts missing available details. NameQuick now reads more sheet context before generating names, so spreadsheet-based prompts have the details they need.
  • Fixed review-worthy prompt failures retrying automatically. Missing or unusable extracted details now stop in Review with evidence instead of repeating the same failed run.
  • Fixed sidebar moves during active processing. Folder drops are now blocked while a rename batch is active, avoiding file moves that can race with ongoing work.
  • Fixed auto-rename notifications opening the wrong place. Show in Finder now targets the renamed file when a smart rename finishes.
