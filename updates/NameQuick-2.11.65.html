Release date: 2026-07-23

New

Improvements

  • Clearer recovery when local AI cannot start. If the MLX runtime is installed but cannot launch, NameQuick now explains the problem and offers a Repair action instead of incorrectly saying the Mac is unsupported.
  • Faster feedback when local AI setup is blocked. Processing now stops promptly and keeps an actionable error in Review when a required local provider or runtime is unavailable, instead of waiting before failing.
  • A better-positioned access check at launch. License and activation progress now stays centered in the startup window.

Bug Fixes

  • Fixed files failing while local AI was still loading at launch. Files added shortly after opening NameQuick are no longer incorrectly rejected before the saved model state finishes loading.
  • Fixed Review showing a badge without a matching item. The Dock badge and Review now use the same list of actionable problems, and processing failures remain available for follow-up.
  • Fixed Presets becoming unresponsive during template editing. Creating, selecting, editing, and saving templates should now remain responsive.
