Release date: 2026-05-30

New

Improvements

  • More reliable Self-Managed license recovery. NameQuick now handles older Self-Managed licenses more consistently after restarting, reinstalling, or signing in again on the same Mac.
  • Clearer device-limit recovery. When a license needs a device slot freed up, NameQuick now keeps the right recovery state visible instead of getting stuck on a loading screen.
  • Safer subscription credit display. Subscription credits now appear only after NameQuick has confirmed the active subscription details, keeping syncing states clear while billing data loads.

Bug Fixes

  • Fixed Self-Managed licenses getting stuck on launch. Older licenses should no longer hang during startup while NameQuick checks activation status.
  • Fixed same-device activation recovery. NameQuick now more reliably recognizes valid activation evidence from the same Mac instead of incorrectly reporting that no device slots are available.
  • Fixed license repair after account changes. Returning from license management now retries the right recovery path and keeps the app state accurate if the device slot still needs attention.
