Release date: 2026-01-14

What's New

  • **Local model downloads in Settings** – Local Providers now list recommended Ollama and LM Studio models with hardware guidance and quick download or copy actions.

Improvements

  • **More reliable local model output** – Local prompts now enforce template rules so assembled filenames match transformed fields, and JSON/fenced responses are cleaned automatically.
  • **Cleaner freeform naming on local providers** – Ollama and LM Studio freeform authoring now uses chat endpoints for more consistent responses.
  • **Suffixes now stick with dot separators.** Smart Rename no longer drops suffix text when you use `.` as the separator.
  • **Smart Rename affixes are now global-only.** Any legacy per-folder prefix/suffix overrides are cleared on upgrade.
